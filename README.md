# mesh
a script for creating wireless mesh network. (B.A.T.M.A.N., 802.11s or simple ad-hoc network wihtout a mesh)

Usage:
```bash
./mesh.sh -h
-h show this help
-c cellid
-n network name
-k channel
-i interface
-b mesh interface
-a (ip address , auto)
-m mode (batman, 802.11s, normal ad-hoc)
```
